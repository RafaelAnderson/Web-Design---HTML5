Diseño Web

Acerca del curso

Aprenderás desde cero a crear de manera rápida y sencilla, páginas web. Dominarás los recursos básicos para iniciarte en el mundo del diseño web, haciendo uso de las principales herramientas para el desarrollo web actual, como Bootstrap y WordPress. Además, estarás capacitado para usar HTML5 y CSS3, los dos lenguajes de programación en los que está basado el desarrollo de contenido web.
